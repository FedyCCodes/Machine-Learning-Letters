Artificial Inteligence Letters

How to use it:

  1. This project will display a letter for you to write.
  2. After that you draw that letter that the project is asking you.
  3. The system will guess what you drew based on data it has collected.

How it works:

  1. It takes the drawn image and stretches or squishes it to fit the corners
  2. Then the stretched image is converted it into a 3 by 3 pixel image.
  3. Once that image is converted, it compares them to it's database.
  4. And checks to see which one looks closest to the drawing.
